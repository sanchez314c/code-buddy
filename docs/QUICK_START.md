# Quick Start

```bash
# Install
git clone https://github.com/sanchez314c/code-buddy.git
cd code-buddy && ./install.sh

# See your portfolio
code-buddy status

# What changed this week
code-buddy pulse

# Which QUEUE repos are ready to ship
code-buddy ready

# Move a repo to ACTIVE
code-buddy move my-app active

# Move a repo out when ACTIVE is full
code-buddy move old-project workbench
```
